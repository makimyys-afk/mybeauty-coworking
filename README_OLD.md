# Бьюти-коворкинг

Веб-приложение для управления бронированиями рабочих мест в бьюти-коворкинге.

## Быстрый старт

```bash
# 1. Установить зависимости
pnpm install

# 2. Настроить базу данных (см. DEPLOYMENT_GUIDE.md)
# Создать файл .env с DATABASE_URL и другими переменными

# 3. Применить схему БД
pnpm db:push

# 4. Заполнить тестовыми данными
node seed-db.mjs
node seed-user-data.mjs
node recreate-reviews.mjs

# 5. Запустить dev-сервер
pnpm dev
```

Приложение будет доступно на http://localhost:3000

## Полная документация

См. файл `DEPLOYMENT_GUIDE.md` для подробной инструкции по развертыванию.

## Технологии

- React 19 + TypeScript
- Tailwind CSS 4
- tRPC 11
- Express 4
- Drizzle ORM
- MySQL/TiDB

## Основные функции

- ✅ Каталог рабочих мест с фильтрацией
- ✅ Система бронирований с QR-кодами
- ✅ Финансовая система с автоматическими транзакциями
- ✅ Страница отзывов от пользователей
- ✅ Административная панель SQL-логов
- ✅ Автоматическая аутентификация (mock-режим)

## Структура

```
├── client/          # Frontend (React + TypeScript)
├── server/          # Backend (Express + tRPC)
├── drizzle/         # Схема БД
├── shared/          # Общие типы
└── *.mjs           # Скрипты заполнения БД
```
